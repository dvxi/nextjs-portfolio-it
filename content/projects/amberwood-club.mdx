Amberwood Club is an early stage React web application that offers a unique e-commerce fragrance shopping experience. App allows clients to create customized fragrance kits tailored to their preferences, season, gender, and more. With smooth animations and a creative concept of product shuffling while answering questions, Amberwood Club aims to provide an engaging and delightful user experience.

## Features

- Personalized Fragrance Kits: Create customized fragrance kits based on individual preferences, season, gender, and other factors.
- Smooth Animations: Enjoy a seamless user experience with smooth animations throughout the app.
- Creative Product Shuffling: Witness an innovative concept of product shuffling while answering questions, making the fragrance selection process fun and interactive.

## Technologies Used

- React: A popular JavaScript library for building user interfaces, providing a robust foundation for the app's frontend.
- Sass: Enhance the styling capabilities of the app using the power of CSS pre-processing with Sass.
- Node.js: Utilize Node.js as the backend runtime environment to handle server-side operations.

## Installation

To run the Amberwood Club app locally, follow these steps:

1. Clone the repository: `git clone https://github.com/dvxi/amberwood-club`
2. Navigate to the project directory: `cd amberwood-club`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`

Note: Make sure you have Node.js and npm installed on your machine.
