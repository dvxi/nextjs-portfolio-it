WordPress site for a physiotherapy clinic, built with Oxygen Builder. Delivers a clear presentation of services, staff, and testimonials; responsive layout and on-page SEO for local visibility.

## Scope

- **Oxygen Builder**: Custom layout and styling without relying on a default theme.
- **Content**: Services, staff profiles, testimonials; structure tuned for both users and search.
- **Responsive and SEO**: Mobile-friendly layout; meta tags and headings aligned with target keywords.
