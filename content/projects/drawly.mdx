Mobile app (React Native, Expo) for the ŚFN (Science Festival in Katowice) 2022: users connect to a 3D printer on the local WiFi, draw on the device, and print in chocolate. Backend: Raspberry Pi with Python server scripts; Azure for connectivity. State and UI: axios, redux, zustand, skia, react-native-navigation.

## Features

- WiFi discovery and connection to a 3D printer on the local network.
- In-app 2D drawing with redo/undo; Skia for rendering.
- Send jobs to the printer; progress tracker and queue for multiple jobs.
- Chocolate-printing pipeline (mechanical part of the project).

![](/drawly-event.jpg)

## Stack and Setup

- **Frontend**: React Native, Expo; Skia for canvas; global/local state with Redux and Zustand.
- **Backend**: Raspberry Pi running Python scripts; Azure for reliable device–server communication.
- **Run locally**: `git clone https://github.com/dvxi/Drawly.git`, then `npm install` and `npm start`; use Expo Go to scan the QR code. Requires WiFi and a compatible 3D printer on the network.
